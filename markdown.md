## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить здездочкой (*
). Например, вот так:
 * элемент 1
 * элемент 2
 * элемент 3

 Чтобы добавить нумерованные списки, небходимо пункты просто пронумеровать. Например, вот так:
 1. Первый пукт.
 2. Второй пункт.
